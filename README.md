# Smart Alarm Clock with Arduino
## Description
A fully functional alarm clock that utilizes a real time clock, LCD display, and multiple buttons to change to set and stop alarms. Built to reinforce strategies of managing multiple components in various states using embedded C++, including non-blocking timing and debounce.
## Features
- Display of real time and date on LCD screen
- Configurable easy-to-use alarm system via buttons
- Buzzer alert which can be turned off manually via button, or automatically after 60 seconds
- Automatic 5 second timeout from set alarm screen if inactivity is detected
## Components Used
- Arduino Uno R3
- RTC DS3231 RTC Module
- 16x2 LCD Display w/ I2C
- Passive Buzzer
- 3 Buttons
